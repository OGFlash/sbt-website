BACKEND API SUMMARY (semp-api — Spring Boot)
────────────────────────────────────────────────────────────────────────────────
Tech Stack:
  - Java / Spring Boot
  - Spring Data JPA + Hibernate
  - MySQL (database: semp_user_data)
  - Spring Mail (Gmail SMTP)

Endpoints:
  POST   /contact           Send contact form email
  GET    /user/all          Get all users
  GET    /user/find/{id}    Get user by ID
  POST   /user/add          Add user
  PUT    /user/update       Update user
  DELETE /user/delete/{id}  Delete user

User Model:
  - id (int, auto-generated)
  - firstName (String)
  - lastName (String)

ContactForm Model:
  - name (String)
  - email (String)
  - message (String)

Notes:
  - CORS allowed from http://localhost:4200 (Angular dev server)
  - Gmail SMTP credentials in application.properties (rotate before reuse)
  - MySQL password stored in plaintext (migrate to env vars / Secrets Manager)
